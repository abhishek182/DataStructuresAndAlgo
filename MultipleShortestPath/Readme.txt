Below is the description of the content of the files:

1. axg137230_Client.java : This file contains the main method. It reads input from Standard Input, creates a graph and insert edges.

2. axg137230_Node.java : This file contains class implementing the abstract data type for the a node in the graph and defines its properties.

3. axg137230_Edge.java : This file contains the class implementing the abstract data type for a weighted edge.

4. axg137230_ShortestPath.java : This file contains the class which provides the abstract data type for the weighted directed graph and implements methods to find number of shortest paths from a given source node to a destination node. 

HOW TO RUN THE PROGRAM:- 
Compile each file using :  $ javac axg137230_Client.java
Run using : $ java axg137230_Client < Input_file
